
# Python Data Analysis Project

This repository contains an academic Python project focused on data creation,
cleaning, exploratory data analysis, and visualisation.

## Project Overview
The project simulates examination data for a tertiary institution and analyses
student performance across age groups, study behaviour, and exam duration.
It also includes analysis of a real-world education dataset to identify
long-term trends.

## Key Tasks
- Creation of structured CSV datasets
- Data cleaning and validation
- Frequency table analysis
- Data visualisation using Matplotlib
- Interpretation of analytical results

## Tools Used
- Python
- NumPy
- Matplotlib
- CSV file handling

## Notes
This project was completed as part of an academic assessment for Python for Data Science.
